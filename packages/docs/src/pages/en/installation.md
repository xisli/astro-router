Ready to install astroid? Follow our automatic set-up guide to get started.


## From the CLI

Run the following command in your terminal based on your package manager.


```shell
# npm
npm i astroid --save-dev

# yarn
yarn add -D astroid

# pnpm
pnpm i astroid --save-dev
```
